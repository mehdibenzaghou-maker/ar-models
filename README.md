# Seb's Garden - One Page (Dark blue + Gold)

Fichiers prêts à télécharger dans un dépôt GitHub Pages.

### Contenu
- `index.html` — page principale (français).
- `styles.css` — thème dark-blue + golden.
- `script.js` — interactions légères.
- `assets/` — (vide) placez vos .glb /.usdz ici si vous voulez lier localement.

### Instructions rapides
1. Déposez tous les fichiers dans la racine du repo GitHub.
2. Activez **GitHub Pages** sur la branche `main` (ou `gh-pages`) et choisissez `root`.
3. Remplacez les URLs des `<model-viewer>` par vos propres fichiers 3D (ex: `assets/plat1.glb` et `ios-src="assets/plat1.usdz"`).
4. Remplissez les champs de contact dans la section Contact.

Les coordonnées ont été laissées **vides** comme demandé.
